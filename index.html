S&P 500 Closing Price Prediction

S&P 500 Closing Price Prediction Using Lasso Regression.

Built a predictive model for forecasting the S&P 500's daily closing price using Lasso Regression. Instead of relying on standard linear regression which struggles with highly correlated stock features (e.g., Open, High, Low prices) Lasso automatically performs feature selection by shrinking less-useful coefficients to zero. This revealed that the day's High price is the dominant predictor of the closing price, allowing the model to achieve strong accuracy while remaining simple and interpretable. The project demonstrates practical understanding of regularization, feature correlation, and model explainability in financial forecasting.

Skills: R | Lasso Regression | Feature Selection | Financial Data Analysis
